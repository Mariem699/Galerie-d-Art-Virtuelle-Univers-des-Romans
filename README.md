# Galerie d'art virtuelle – L'univers des romans

## Description
## Description
Bienvenue à la Galerie d’Art Virtuelle, une plateforme web dédiée à la découverte du monde des romans.
Ce projet permet aux utilisateurs d’explorer différents types de livres à travers une interface simple, moderne et agréable.  
L’objectif est de créer une expérience immersive où l’utilisateur peut naviguer facilement entre plusieurs univers littéraires.



## Catégories disponibles
Le site web propose plusieurs genres littéraires de romans, tels que :
- Romance
- Fantastique
- Science-fiction
- Autobiographie
- Littérature française

## Technologies utilisées
- HTML5
- CSS3

## Fonctionnalités
- Navigation fluide et intuitive
- Classification des romans
- Interface claire et bien organisée

## Objectif du projet
Ce travail, réalisé dans le cadre d'un apprentissage du développement web, vise à :
- maîtriser les bases de HTML5 et CSS3
- organiser une page web
- créer une interface conviviale
